#autoFNaF

an open-source hotkey system for FNaF

hotkeys:
q: left door
a: left light
e: right door
d: right light
s: camera
r + 1 - 0 / t: cameras
z: look left
x: look middle / recalibrate
c: look right
m: new game
n: continue
b: 6th night
v: custom night
f / g: change freddy difficulty
h/ j: change bonnie difficulty
k / l: change chica difficulty
o / p: change foxy difficulty
i: ready
